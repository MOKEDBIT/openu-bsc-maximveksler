דוגמא 1

הנה השיטה לסיכום איברי המערך בצורה איטרטיבית (לולאתית) ובצורה  רקורסיבית (קצת שונה מהמוצג בהרצאה)

public static int sumArray (int[] values)

{

    int sum = 0;

    for (int count = 0; count < values.length; count++)

         sum += values[count];

    return sum;

}

      

public static int recSum (int [] a, int i)

{

    if (i==a.length)

          return 0;

    return (a[i] + recSum (a, i+1));

}

            

דוגמא 2

להלן שיטה לחיפוש איבר x במערך של מספרים שלמים. שוב, באופן איטרטיבי ובאופן רקורסיבי:

 

public static boolean find (int[] arr, int x)

{

    for (int i=0; i<arr.length; i++)

      if (arr[i] == x)

            return true;

    return false;

}

 

public static boolean recFind (int []a, int i, int x)

{

    if (i== a.length)

           return false;

    if (a[i] == x)

           return true;

    return recFind (a, i+1, x);

}

    

 

דוגמא 3

הנה שיטה למציאת מקסימום במערך של מספרים שלמים (הפעם רק השיטה הרקורסיבית):

    

public static int recMax (int []a, int i)

{

    if (i==a.length-1)

           return a[a.length-1];

    int currentMax = recMax (a, i+1);

           if (a[i] < currentMax)

              return currentMax;

           else

              return a[i];

}

 

 

 

דוגמא 4

השיטה הבאה מקבלת מערך מלא במספרים שלמים ובודקת אם כל אחד מהאיברים במערך גדול או שווה לסכום כל האיברים שלפניו במערך.

הפעם אנו משתמשים בשיטה שהוצגה בהרצאה לסיכום רקורסיבי של איברי המערך.

public static int recSumArray (int [] a, int lo, int hi)

{

     if (hi < lo)

         return 0;

     return a[lo] + recSumArray (a, lo+1, hi);

}

 

public static boolean check (int [] a, int i)

{

     if (i== a.length)

          return true;

     if (a[i] < recSumArray (a, 0, i-1))

          return false;

     return check (a, i+1);

}
