דוגמא 1

מה מבצעת השיטה הבאה?

int what (int z, int w)

{

   int i, j;

 

   if (w == 0)

     return 0;

   else

   {

     i = 1;

     j = z;

     while (2*i <= w)

     {

        i = 2*i;

        j = 2*j;

     }

     return (j + what(z, w-i));

   }

}

 

תשובה לדוגמא 1- השיטה what  מבצעת כפל של שני מספרים שלמים.

דוגמא 2

מה יודפס בפלט לאחר הרצת התכנית הבאה? האם אתה יכול להכליל את תשובתך?

Class A {

   public static void main(String [] args)

   {

      System.out.println (f(7, 3));

      System.out.println (f(7, 4));

   }

 

   public static int f (int m, int n)

   {

      if (m == n)

         return m+1;

      return (f (m, f(m-1, n+1)));

   }

}

דוגמא 3

לפניך חמש שיטות רקורסיביות. מה מבצעת כל אחת מהן (באופן כללי):

int f1 (int n)

{

     if (n == 0)

           return 0;

     else

           return n * f1(n-1);

}

 

float f2 (float x, int n)

{

     if (n == 0)

           return 0;

     else

           return x + f2(x, n-1);

}

 

int f3(int n)

{

     if (n < 2)

           return 0;

     else

           return 1 + f3(n/2);

}

 

int f4(int n)

{

     if (n == 0)

           return 0;

     else  return f4(n/10) + n%10;

}

 

int f5(int n)

{

     if (n < 0)

           return f5(-n);

     else if (n < 10)

           return n;

     else return f5(n/10);

}

 

תשובה לדוגמא 3

 

f1 תחזיר תמיד 0, כיוון שהגורם האחרון שיוחזר הוא 0, והוא תמיד משתתף במכפלה. אם נשנה את הערך המוחזר בתנאי הסיום ל - 1, הפונקציה תחזיר את ערך העצרת של n (n!)

f2 תחזיר את המכפלה של שני הפרמטרים.

f3 תחזיר את הערך השלם של log2n

f4 תחזיר את סכום הספרות במספר.

f5 תחזיר את הספרה השמאלית ביותר של מספר.
