הדוגמא הבאה מתייחסת למציאת מחלק משותף מקסימלי (ממ"מ) של שני מספרים טבעיים. נגדיר את c להיות המחלק המשותף המקסימלי (GCD - greatest common divisor) של שני מספרים a ו- b אם c מחלק את a ואת b ואין אף מספר אחר שמחלק אותם שהוא יותר גדול מ- c. למשל, המספרים 2, 7 ו- 14 מחלקים את 28 ואת 70. ולכן, 14 הוא הממ"מ שלהם.

נתאר כעת את האלגוריתם של אויקלידס (Euclid) למציאת הממ"מ. זהו למעשה האלגוריתם הלא-טריוויאלי הראשון המפורסם.

ובכן, לטענת אויקלידס, הממ"מ של שני מספרים שווה לממ"מ של המספר הקטן מבין השניים ושל ההפרש ביניהם. אנו לא נוכיח טענה זו כאן, אך נשתמש בה באלגוריתם. ברור שאם שני המספרים a ו-b זהים, גם הממ"מ שווה להם. זה יהיה גבול הרקורסיה. אם אחד מהם גדול מהשני, הממ"מ יהיה שווה לממ"מ של המספר הקטן ושל ההפרש בין השניים, ולכן נמשיך לחפש את הממ"מ של המספרים הבאים, שבהכרח יותר קטנים מהמספרים המקוריים. זהו צעד הרקורסיה, שמתקדם לכוון תנאי העצירה.

בדוגמא שהבאנו נבדוק מהו הממ"מ של 28 ו- 70. מהם נעבור לממ"מ של 28 (הקטן) ו- 42 (ההפרש). מהם נעבור לממ"מ של 28 (הקטן) ו- 14 (ההפרש). מהם נעבור לממ"מ של 14 (הקטן) ו- 14 (ההפרש). 14 ו- 14 שווים, ולכן גם הממ"מ שלהם הוא 14. לכן הממ"מ של 28 ו- 70 הוא 14. 

להלן השיטה gcd שמבצעת את האלגוריתם:

static int gcd (int n, int m)

{

   if (n == m)

       return n;                     // stop condition

   if (n < m)

       return (gcd (n, m-n));        // recursive calls

   return (gcd (m, n-m));

}
