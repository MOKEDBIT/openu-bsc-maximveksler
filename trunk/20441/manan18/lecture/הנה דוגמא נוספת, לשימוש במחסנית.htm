הנה דוגמא נוספת, לשימוש במחסנית:

השיטה הבאה valueAtPosition מקבלת כפרמטר מספר שלם k ומחזירה את ערכו של האיבר הנמצא בעומק k במחסנית. השיטה אינה משנה את מצבה הסופי של המחסנית (כלומר היא לא מוציאה את האיבר מהמחסנית אלא רק בודקת מה ערכו).

לדוגמא, אם המחסנית נראית כך:

12

4

6

20

5

 

אזי הקריאה ל- valueAtPosition (4)  תחזיר את הערך 20. אם במחסנית פחות מ- k איברים, יוחזר הערך -9999.

 

אנו נשתמש במחסנית עזר לצורך הפתרון.

 

public int valueAtPosition (int pos)

{

   DataStack st = new DataStack();

   for (int i = 0; i<pos-1; i++)

   {

       if (empty())

           return -9999;

       st.push(this.pop());

   }

     

   DataNode item = top();

   int data = item.getValue();

   for (int i = 0; i<pos-1; i++)

        this.push (st.pop());

   return data;

}
